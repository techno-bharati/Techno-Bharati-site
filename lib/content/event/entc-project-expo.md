## Rules

- Participants must carry a **valid College/University ID card**
- Registration is not accepted without showing the **fee receipt**
- **Team size:** Minimum 3 — Maximum 4 participants
- Organizers will provide only **230V AC mains** electric power supply
- Every team must **report to the venue on time**
- Each team gets **8–10 minutes** for Project Presentation and **5 minutes** for Q&A
- Judge's decision is **final and binding**
- Project must be an **innovative working model** or **hardware + software based** project
- All registered participants will receive **certificates**
