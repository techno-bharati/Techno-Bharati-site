## Rules
- Participants must carry a **valid College/University ID card**
- Report **30 minutes before** event time for Round 1
- Registration is not accepted without showing the **fee receipt**
- **Team size:** Maximum 2 participants
- Contest will be conducted in **two rounds**
- Judge's decision is **final and binding**
- All registered participants will receive **certificates**

## Format

##### Round 1 — Tech Dangal Quiz *(MCQ Round)*
- Topics: **Basics of C, C++, Python**
- Time limit: **30 minutes**
- **Top 5 groups** shortlisted for Round 2

##### Round 2 — Code Dangal *(Speed + Logic Coding)*
- Allowed languages: **C / C++ / Python**
- **Internet not allowed**
- Time limit: **30 minutes**
- Winner decided by **output correctness + execution time**
